## 📌 Sobre o Projeto

Este projeto apresenta um Dashboard de Produção desenvolvido em Power BI, criado para monitorar e analisar indicadores essenciais de desempenho operacional.

O objetivo é fornecer uma visualização clara e interativa dos dados de produção, permitindo que gestores e analistas acompanhem produtividade, qualidade, disponibilidade e desempenho ao longo do tempo.

O dashboard facilita a tomada de decisões baseada em dados, identificando padrões, quedas de produção e oportunidades de melhoria.

🎯 Principais Indicadores

O painel apresenta os seguintes KPIs principais:

📦 Total Aprovado
Quantidade total de itens produzidos aprovados.

❌ Total Rejeitado
Total de itens que não passaram no controle de qualidade.

⏱ Horas Produtivas
Tempo total de produção efetiva.

🛑 Horas Paradas
Tempo de parada da operação.

📈 Visualizações do Dashboard

🎛 Filtros Interativos

O dashboard conta com filtros dinâmicos que permitem análises específicas:

👷 Operador

📅 Mês

Esses filtros permitem analisar o desempenho individual ou por período.

🧠 Insights que o Dashboard Permite

Com este painel é possível identificar:

períodos de queda na produção

operadores com melhor desempenho

impacto das paradas na produção

níveis de qualidade do processo

tendências de produtividade ao longo do ano
